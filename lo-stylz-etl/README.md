# Lo Stylz Salon Employee ETL

This ETL process extracts employee records from `employees_source.csv`, transforms the data into a clean reporting format, and loads the result into `employees.csv`.

## Files
- `employees_source.csv` - Source data input file.
- `etl_employees.py` - ETL script.
- `employees.csv` - Output file generated by the ETL script.

## Run
From this folder, run:

```bash
python etl_employees.py
```

After running, `employees.csv` will be created in the same folder.
